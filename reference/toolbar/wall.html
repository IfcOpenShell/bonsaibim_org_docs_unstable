Create Wall¶

The Create Wall tool in Bonsai allows you to add and manipulate wall elements in your floor plan. It provides options for specifying wall element properties and adjusting their alignment and orientation.

Toolbar Icon¶

Tool Options¶

Shortcut¶

To access the Create Wall tool, you can use the following keyboard shortcut sequence:

Shift + Spacebar, ???

Key Features¶

Add Wall¶

  • Shortcut: Shift + A

  • Adds a new wall segment at the 3D cursor location.

Add or Modify Wall¶

  • Height: Set the wall height.

  • Length: Set the wall length.

  • X Angle: Adjust the angle of the wall.

Modify Wall¶

  • Extend (Shortcut: Shift + E): Extend an existing wall to another face.

  • Butt (Shortcut: Shift + T): Join wall segments end-to-end.

  • Mitre (Shortcut: Shift + Y): Create a mitre joint between two wall segments.

  • Merge (Shortcut: Shift + M): Merge two wall segments into one.

  • Flip (Shortcut: Shift + F): Flip the direction of the wall segment.

  • Split (Shortcut: Shift + K): Split a wall segment into two parts.

  • Rotate 90 (Shortcut: Shift + R): Rotate the wall by 90 degrees.

  • Regen (Shortcut: Shift + G): Regenerate the wall geometry.

  • Add Void (Shortcut: Shift + O): Create an opening in the wall for doors, windows, or other elements.

Change walls alignment or move walls¶

  • Align Exterior (Shortcut: Shift + X): Align wall to the exterior face.

  • Align Centerline (Shortcut: Shift + C): Align wall to the centerline.

  • Align Interior (Shortcut: Shift + V): Align wall to the interior face.

  • Mirror (Shortcut: Shift + M): Mirror wall.

Wall Mode¶

  • Void (Shortcut: Alt + O): Create voids in the walls.

  • Decomposition (Shortcut: Alt + D): Decompose wall elements.

Aggregation¶

  • Assign (Shortcut: Ctrl + P): Assign wall elements to a specific group.

  • Unassign (Shortcut: Alt + P): Remove wall elements from a group.

Calculate Quantities¶

  • Calculate All Quantities (Shortcut: Q): Calculate all quantities related to the walls (e.g., area, volume).

Usage¶

To use the Create Wall tool:

  1. Activate the tool from the toolbar or use the shortcut A.

  2. Select the wall type from the dropdown menu, or create a new wall type (which represents the wall thickness) and set other desired parameters.

  3. Set the 3D cursor to the desired starting location for the wall by holding Shift and left-clicking in the 3D viewport.

  4. Add the wall by pressing Shift + A.

  5. Adjust the length of the wall segment by dragging the “Length” parameter or entering a numeric value.

  6. Set the 3D cursor to the new location for the next wall segment, enabling snapping to ensure the segments butt together properly.

  7. Add the next wall segment by pressing Shift + A again.

  8. Rotate the new wall segment if needed by pressing Shift + R and adjusting the angle.

  9. Adjust the length of the new segment as required.

  10. Repeat steps 6-9, setting the 3D cursor to the desired location, adding new wall segments, rotating, and adjusting lengths, until the complete wall perimeter is created.

  11. Modify wall segments using the various shortcuts (e.g., extend, butt, mitre, merge, flip, split, rotate) as needed to refine the wall layout.

  12. Align and adjust walls using the alignment options.

  13. Add voids by clicking “Add Void” and placing openings where necessary.

  14. Calculate quantities by pressing Q to ensure accurate measurements.

The Create Wall tool in Bonsai enables precise and flexible wall creation and management, enhancing your ability to develop detailed floor plans efficiently.
